What is this page about?
You will find 3 games here. All have their own different mechanics and are provided with detailed descriptions about their rules and objectives. Have fun playing! ♥
Game 1
Your task is to break a secret 4-digit code consisting of digits from 0 to 9. The code is randomly generated at the start of each game. You have 10 attempts to guess the code. The numbers in code do not repeat. After each attempt, you'll receive feedback: how many digits are in the correct position (correct), and how many digits are in the code but in the wrong position (present).
